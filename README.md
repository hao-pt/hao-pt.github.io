# hao-pt.github.io

Personal academic website built with [al-folio](https://github.com/alshedivat/al-folio) (Jekyll).

## Local Development

Requires [Ruby](https://www.ruby-lang.org/en/downloads/), [Bundler](https://bundler.io/), and [Python](https://www.python.org/) with pip.

```bash
# Install dependencies (first time only)
bundle install
pip install jupyter

# Start dev server
bundle exec jekyll serve
```

Site runs at http://localhost:4000.

For deployment details, see [INSTALL.md](INSTALL.md).
